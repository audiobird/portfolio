The MRCC 4x4

Prototype ready for software development

The 4x4 is Conductive Lab's MRCC's little brother. It controls four din and virtual inputs and outputs. A very versatile midi router interface.

The MRCC 4x1

Beta testing unit and software dev prototype pictured.

The 4x1 is an input expander for Conductive's MRCC. It handles four din inputs and four virtual outputs.
